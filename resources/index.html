Resources, software and tools

Public tools, methods and infrastructure connected to movement ecology, demography and animal-borne monitoring.

This page collects selected resources that are useful for students, collaborators and applied partners. It distinguishes group software and infrastructure from external tools that are widely used in movement ecology and ecological modelling.
